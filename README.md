# 强势回踩系统

主板强势股 5 日线回踩纪律工作台。

系统定位：本地交易纪律与复盘工具。不连接券商，不自动下单，不构成投资建议。

## 系统功能

- 股票池分析：只看沪深主板 A 股，按成交额取当前批次前 30。
- 待买观察：强势股回踩 MA5 不破才进入买点观察。
- 交易记录：买入、卖出流水是核心数据源。
- 自动持仓：由交易记录按移动平均成本法推导当前持仓。
- 自动资产：由交易记录计算现金、持仓市值、总资产、盈亏。
- 交易复盘：审计每笔买入卖出是否符合强势回踩规则。
- 日报周报月报：自动汇总交易，并支持手写心得。

## 股票范围

只保留：

```text
600、601、603、605
000、001、002
```

排除：

```text
ST / *ST
300、301 创业板
688、689 科创板
8、4、920 北交所
000725 京东方A
```

自动股票池每次生成都会替换当前初筛池，旧批次只归档，不会一直累积。

## 日常流程

盘前：

1. 在「数据导入」自动生成或上传股票池。
2. 系统过滤主板并保留成交额前 30。
3. 一键分析并补历史K线。
4. 查看「今日看板」「盘中观察」「待买」。

盘中：

1. 在同花顺、券商软件或模拟盘手动完成买入/卖出。
2. 在「交易记录」录入或编辑交易流水。
3. 系统自动更新持仓、现金、总资产、盈亏。

收盘后：

1. 在「持仓监控」检查 MA5 风控提醒。
2. 在「资产看板」查看资产和风险暴露。
3. 在「交易复盘」查看规则审计。
4. 在「报告中心」填写日报。

周末：

1. 查看周报。
2. 编辑心得。
3. 总结违规交易和下周计划。

## 交易记录驱动持仓

交易流水保存到：

```text
data/trades/trade_log.csv
```

字段包括：

```text
代码、名称、类型、日期、时间、价格、数量、金额、手续费、印花税、原因、备注
```

买入后：

```text
现金 = 现金 - 成交金额 - 手续费
持仓数量增加
平均成本重新计算
```

卖出后：

```text
现金 = 现金 + 成交金额 - 手续费 - 印花税
持仓数量减少
已实现盈亏按移动平均成本计算
```

持仓数量为 0 的股票不会出现在当前持仓里，但会保留在交易复盘中。

## 资产计算

```text
当前现金 = 初始本金 - 买入支出 + 卖出收入
持仓市值 = 当前持仓数量 x 当前价
当前总资产 = 当前现金 + 持仓市值
总盈亏 = 当前总资产 - 初始本金
总收益率 = 总盈亏 / 初始本金 x 100%
```

当前价来源优先级：

1. 今日股票池现价
2. 盘中刷新行情
3. 历史K线最后收盘价
4. 用户在交易记录中手动修正

## 报告生成

报告中心支持：

- 日报：`data/reports/daily/`
- 周报：`data/reports/weekly/`
- 月报：`data/reports/monthly/`

当日没有买入卖出时，不生成交易分析，但仍可记录观察和心得。
