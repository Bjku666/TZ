# 强势回踩短线交易纪律系统

主板强势股 5 日线回踩纪律工作台。

当前版本：`v2.1 Discipline Desk`

当前规则口径：`股票池分组规范 v3`

系统定位：本地交易纪律与复盘工具。不连接券商，不自动下单，不构成投资建议。

## 本版更新

- 参考前端版重做视觉风格：深色纪律导航、浅色工作区、紧凑卡片、青蓝动作色和红绿交易语义。
- 今日看板增加纪律概览条，集中展示待买确认、缺K线、风险持仓和未复盘交易。
- 侧栏增加事件流，记录刷新、股票池分析和交易流水保存等关键动作。
- 股票分组页增加代码/名称搜索，便于快速定位股票。
- 初筛池生成、行情刷新、成交额前30变动扫描拆成独立动作；刷新行情不会替换今日锁定池。

## 系统功能

- 股票池分析：只看沪深主板 A 股，按成交额取当前批次前 30。
- 待买观察：强势股回踩 MA5 0%-2% 且不破才进入待买组。
- 交易记录：买入、卖出流水是核心数据源。
- 自动持仓：由交易记录按移动平均成本法推导当前持仓。
- 自动资产：由交易记录计算现金、持仓市值、总资产、盈亏。
- 交易复盘：审计每笔买入卖出是否符合强势回踩规则。
- 日报周报月报：自动汇总交易，并支持手写心得。

## 股票范围

只保留：

```text
600、601、603、605
000、001、002
```

排除：

```text
ST / *ST
300、301 创业板
688、689 科创板
8、4、920 北交所
000725 京东方A
```

自动生成或同花顺上传会创建新的今日初筛池批次，并记录批次、来源、生成时间和生成时排名。行情刷新只更新当前池内股票的现价、涨跌幅、成交额、MA5偏离率和分组状态，不增删初筛池名单。

成交额前30扫描只做提醒：

```text
新进前30
跌出前30
排名上升
排名下降
```

扫描不会自动覆盖当前初筛池；只有重新生成、重新上传或手动纳入新进股票，才会改变名单。

## 股票池分组规范 v3

系统只维护三个股票池分组：

```text
初筛
观察
待买
```

三组是逐层筛选关系：

```text
待买 ⊆ 观察 ⊆ 初筛
```

核心规则：

```text
初筛 = 合规主板 A 股中按成交额排序取前 30
观察 = 初筛里已有 5% 以上阳线启动、未放量破位、未有效跌破 MA5
待买 = 观察里 MA5 偏离率 0%-2% 且未有效跌破 MA5
```

MA5 偏离率状态：

```text
D5 < 0%      → 跌破MA5，不进入待买
0% ≤ D5 ≤ 2% → 待买观察，进入待买组
2% < D5 ≤ 5% → 继续观察，等待回踩
5% < D5 ≤ 7% → 偏高不追
D5 > 7%      → 远离不追
```

统一流程阶段：

```text
初筛通过、强势确认、继续观察、偏高不追、远离不追、待买观察、跌破MA5、风险排除、未达规则、淘汰
```

阶段到分组映射：

```text
初筛通过、未达规则、淘汰、跌破MA5、风险排除 → 初筛
强势确认、继续观察、偏高不追、远离不追 → 观察
待买观察 → 待买
```

旧命名兼容：

```text
接近买点 → 待买观察
等回踩 → 继续观察
```

本金、可用现金、一手金额只用于资产展示、交易记录和现金检查，不参与观察/待买分组判断。

## 日常流程

盘前：

1. 在「数据导入」自动生成或上传股票池。
2. 系统过滤主板并保留成交额前 30。
3. 一键分析并补历史K线。
4. 查看「今日看板」「盘中观察」「待买」。

盘中：

1. 点击「刷新当前池行情」更新已锁定名单内的行情和分组。
2. 点击「扫描新进前30」查看成交额榜变化，必要时手动纳入。
3. 在同花顺、券商软件或模拟盘手动完成买入/卖出。
4. 在「交易记录」录入或编辑交易流水。
5. 系统自动更新持仓、现金、总资产、盈亏。

收盘后：

1. 在「持仓监控」检查 MA5 风控提醒。
2. 在「资产看板」查看资产和风险暴露。
3. 在「复盘报告」查看规则审计。
4. 在「复盘报告」填写日报。

周末：

1. 查看周报。
2. 编辑心得。
3. 总结违规交易和下周计划。

## 交易记录驱动持仓

交易流水保存到：

```text
data/trades/trade_log.csv
```

字段包括：

```text
代码、名称、类型、日期、时间、价格、数量、金额、手续费、印花税、原因、备注
```

买入后：

```text
现金 = 现金 - 成交金额 - 手续费
持仓数量增加
平均成本重新计算
```

卖出后：

```text
现金 = 现金 + 成交金额 - 手续费 - 印花税
持仓数量减少
已实现盈亏按移动平均成本计算
```

持仓数量为 0 的股票不会出现在当前持仓里，但会保留在交易复盘中。

## 资产计算

```text
当前现金 = 初始本金 - 买入支出 + 卖出收入
持仓市值 = 当前持仓数量 x 当前价
当前总资产 = 当前现金 + 持仓市值
总盈亏 = 当前总资产 - 初始本金
总收益率 = 总盈亏 / 初始本金 x 100%
```

当前价来源优先级：

1. 今日股票池现价
2. 盘中刷新行情
3. 历史K线最后收盘价
4. 用户在交易记录中手动修正

## 报告生成

复盘报告支持：

- 日报：`data/reports/daily/`
- 周报：`data/reports/weekly/`
- 月报：`data/reports/monthly/`

当日没有买入卖出时，不生成交易分析，但仍可记录观察和心得。
