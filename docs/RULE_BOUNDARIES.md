# 规则边界

系统所有提示分为三类。

## 视频原版交易信号

- 正式收盘成交额原始前 20。
- 市场范围和 ST 等基本排除。
- 入选日收盘站上 MA5。
- 下一交易日起等待 MA5 回踩。
- 视频买入时段内位于 MA5 回踩区。
- 隔日超短卖出提醒。

## 交易所和账户执行约束

- 100 股一手。
- 现金不足。
- T+1 可卖数量。
- 停牌、跌停、非交易时段。
- 手续费、印花税、过户费。
- 人工确认。

这些约束会影响 `executionAllowed`，但不得改写 `signalQualified`。

## 工程口径

- MA5 回踩容差 `0.5%`。
- 行情新鲜度 `20` 秒。
- 正式批次生成时间建议 `15:05` 后。
- 10 点涨停后采用最小补全。

工程口径不是视频作者额外说过的策略条件。
