# Account Workspace V3

## 定位

TZ V3 是五日线回踩隔日超短策略的执行纪律、账户记录和复盘工具，而不是行情终端或选股系统。

## 数据边界

### 独立账户数据

以下实体都必须带 `mode`：

- 交易流水；
- 账户设置与初始资金；
- 持仓推导与可卖数量；
- 延迟至尾盘决定；
- 持仓备注；
- 通知；
- 日、周、月复盘；
- 对账数字。

### 不接入的内容

- 自动选股、股票池、前 20 排名；
- 盘中全市场扫描；
- 实时行情源；
- 券商登录、自动同步或自动下单；
- 自动荐股与买卖建议。

## 复盘模型

每条复盘记录包含四项：

1. `planAndBasis`：计划与交易依据；
2. `executionAndDeviation`：执行与纪律偏差；
3. `resultAndEmotion`：结果归因与情绪复盘；
4. `improvementAndNextPlan`：改进与下一交易日硬计划。

## 持仓计算

持仓由当前账户交易流水按时间顺序推导：

- 买入增加批次；
- 卖出按 FIFO 扣减；
- 当日买入数量计入 T+1 锁定；
- 可卖数量只包含买入日期早于当前日期的剩余批次；
- 最近一笔成交价作为无实时行情时的参考价格；
- 同花顺手工对账只改变展示的期末数字，不修改交易流水。
